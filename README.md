## Installatie
Klik op de groene knop met "Code" in de rechterbovenhoek van de pagina.
```
$ <> Code
```

Selecteer "Download Zip" om de repository als een Zip- bestand te downloaden.
```
$ Download ZIP
```

Open de gedownloade Zip- map, pak de vervolgens de inhoud uit naar een locatie op jouw computer.

```
$ Bedieningspaneel.Zip
```

Open het uitgepakte bestand en navigeer naar het bestand met de naam "index.html", open deze in jouw favoriete websitebrowser om de website te bekijken.

```
$ Index.html
```
